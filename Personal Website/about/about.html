More Detail:

I started learning about Python, then discovered the Raspberry Pi, which is a small personal computer that has input and output pins on it so it can easily be used to connect to and control devices/hardware. I had fun using the Raspberry Pi, my first project was a room alarm. The room alarm just used a motion detector that was connected to the Raspberry Pi. The program would be started when I left the room and take a picture or video, using a camera connected to the Raspberry Pi every time motion was detected. This got me very interested in programming, so I decided to take some programming classes and ultimately pursue a computer science degree. The first class I took was a C++ class, and it really got me more interested in programming. After that first C++ class I took a Visual Basic class which was fun too. After taking the Visual Basic class I bought an Arduino with a temperature and humidity sensor and wrote a program to display the data and chart it in Visual Basic. I was really having fun. As I became more interested in Arduino's and sensors, I learned that there is an Arduino board with a built in Wi-Fi and Bluetooth chip, the ESP32, so it could wirelessly connect to a network. This got me into learning about the “Internet of Things” or IoT and I then bought about 5 of these Wi-Fi Arduino boards to log temperature, humidity, and a few other things in every room of my home. That data was useless unless I had a platform to see it on, so I started learning about Message Queuing Telemetry Transport, or MQTT and shortly after created a home MQTT network that received all the data that my devices were sending to it, around 80 data points per minute. I learned about Node-Red and built a graphical user interface or GUI so I could see all of my data that was being sent. For this, a laptop, which is all I had, wasn't the best solution, so I built a Linux computer, with the Ubuntu distrobution, to be used as the MQTT server and to get familiar with Linux. I learned more and more about Node-Red, but after a few days of all the data it was storing and displaying on graphs, it would slow it way down and it would be hard to use. I needed a database to store all this information I was collecting so it wasn’t bogging down my system, for that I set up an SQL database using MariaDB (a free version of MySQL) and used phpMyAdmin to access the data and to learn about MySQL and how to query data. I wrote a program (~900 lines of code) in ardino code (C/C++) and uploaded it to each Arduino Wi-Fi board and use Node-Red to receive the data sent from the Wi-Fi Arduino boards, then Node-Red would also send that data to the SQL DB where I could query it or send it to an excel file to look at it. The Node-Red GUI was nice, but I then learned about "Home Assistant" which is similar to Node-Red, but has some extra features (it's technically an OS) for home automation and also has, in my opinion, a nicer more user friendly GUI. Since it is an OS and I didn’t want to dedicate an entire computer to it, I learned about and created a virtual machine, or VM, to run it on. There is less programming to get Home Assistant set up and working, but it can communicate with Node-Red for certain things and be programmed to run automations. Then I became interested in HTML, CSS, and JavaScript so I went through a bunch of internet tutorials, including most of the ones that Khan Academy offered. That was fun, but I didn’t really have too many applications to use it with. I then bought some Wi-Fi connected AC outlet smart switches and connected those to my home automation and have them turn on when the light value reaches a certain level. They are connected to Alexa so they can be verbally turned on or off as well. I set up some home automations that will send a push notification to my wife when arrive at work, leave work, and get home. I also set up an automation that senses motion on our patio, and if it’s dark outside will turn on the inside lights if they’re off, start recording on an IP camera on the patio, and send me a push notification, if inside lights are on it sends a push notification to our smart TV. Since this program runs on a local server, I’ve set up port forwarding and a DNS so I can access it when not connected to my home network. I recently started learning about “robot operating system“ or ROS and that will be used for future robotic projects, which is something else that really interest me. Things like self-driving RC cars or self-driving drones and programming robotic arms to do things. I’d also like to get a 3D printer to make custom things for my projects. I’ve also recently been learning about web development and earned a few certificates in the process. I took a Javascript Skill Assesment Test on LinkedIn and scored in the top 15% out of over 1,000,000(as of 5/1/2021) people who took it, I also took a CSS Skill Assesment Test on LinkedIn and scored in the top 15% as well out of 872,000(as of 5/1/2021) people who took it.

Health and Fitness:

As a kid growing up, I was always skinny. I ate a lot and was always outside playing and being active because it was fun. I would mostly ride my bike all around town, but some other things that were fun for me were to rollerblade and play roller hockey, jump on our big trampoline, Karate, play basketball, baseball, football and some other thingas that aren't coming to mind at the moment. Once I got my Drivers license, I was having so much fun driving everywhere that most of the physical activities stopped except for some weight training, but the eating a lot stayed the same. This was fine until about the age of 19, which is when I think my metabolism started to slow down, and I noticed something I had never before seen on myself, I was starting to get a "gut". I didn't think to much of it, and after a few years went by I got a new job and an old friend saw me and made a comment that I had gained weight since he last saw me a few years prior. That's when I took a good look in the mirror and realized that I had put on some weight. I thought working out more would just make it go away, but after doing some weight training at the gym, my muscles were growing, but my excess fat wasn't leaving. I wasn't sure what to do. Then a co-worker of mine told me about a book written by Tim Ferris called "The 4 Hour Body" a book described as "focusing on diet, with chapters on exercise, sleep, and sexual performance.". It is a very interesting book, I was interested in it for the diet part, what Tim refers to as the "Slow Carb Diet", descripbed simply as: "eat a very simple set of meals repeatedly; focus on "slow carbs"; and allow one "cheat day" a week when all foods are allowed. The diet prohibits starches and all things sweet (including fruit and all artificial sweeteners), with strong preferences for lean protein and a few specific vegetables." There are of course more details in his book, but thats the gist of it. Some of my co-workers and I started a weight loss challenge, most of us following Tim's the slow carb diet. This got me more interested in diet and excercise and how the two help to burn fat. I realized that burning fat, or losing weight, was may more about doet than it was excercise. After a few months I went from 220Lbs to 170Lbs, which is what I felt was an ideal weight for my body. Once I lost the weight, I knew it would come right back if I got back into my old habbits, so I tried to focus on eating healthy and healthy proportions, which wasn't always easy. I've gotten into bad habbits and put weight on since, but when I focus on back on eating healthy and eating healthy proportions, I am able to get back to a healthy weight. Although I have noticed as I've been getting older, it takes a little longer for the weight to come off.
