Introduction

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

Python is used for:
  • Web development (server-side)
  • Software development
  • Mathematics
  • System scripting
  • On a server to create web applications.
  • Alongside software to create workflows.
  • To connect to database systems. It can also read and modify files.
  • To handle big data and perform complex mathematics.
  • Rapid prototyping, or for production-ready software development.
Python vs Other Lanuguages
  • Python was designed for readability, and has some similarities to the English language with influence from mathematics.
  • Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.
  • Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Syntax

As we learned in the previous section, Python syntax can be executed by writing directly in the Command Line:

>>> print("Hello, World!")

Output: Hello, World!

Or if using an IDE, depending on the IDE, you can also run one line of code at a time, or write a program and execute it.

Indentation

Indentation refers to the spaces at the beginning of a code line. Where in other programming languages the indentation in code is for readability only, the indentation in Python is very important. Python uses indentation to indicate a block of code.

if 1 < 2:
    print("2 is greater than 1")

Output: 2 is greater than 1

Without using indentation, the above statement would print out "2 is greater the 1" whether it's true or not.



Comments
  • Comments can be used to explain Python code.
  • Comments can be used to make the code more readable.
  • Comments can be used to prevent execution when testing code.
Single Line Comments

Comments starts with a #, and Python will ignore them:

#This is a single line comment, this part of the code will be ignored by the compiler
print("Hello friend")

It is good practice to use comments to explain your code so other readers can understand it easier and so can you can better understand what you wee doing if you haven't looked at the code for a while. You can either put comments directly above or after the code you're describing.

print("Hello friend") #This is a comment and will be ignored by the compiler
Multiline Comments

Python does not really have a syntax for multi line comments. To add a multiline comment you could insert a # for each line:

#This is a multiline
#comment and it will
#be ignored by the compiler
print("Hello friend")

Or, not quite as intended, you can use a multiline string. Since Python will ignore string literals that are not assigned to a variable, you can add a multiline string (triple quotes) in your code, and place your comment inside it:

"""
This is a multiline
comment and it will be
ignored by the compiler
"""
print("Hello friend")

Booleans

Booleans represent one of two values: True or False. In programming you often need to know if an expression is True or False. You can evaluate any expression in Python, and get one of two answers, True or False. When you compare two values, the expression is evaluated and Python returns the Boolean answer:

print(10 > 9)
print(10 == 9)
print(10 < 9)

Output: True
Output: False
Output: False

Print a message based on whether the condition is True or False:

a = 50
b = 20

if b > a:
    print("b is greater than a")
else:
    print("b is not greater than a")

Output: b is not greater than a


Lists

Lists are used to store multiple items in a single variable. Lists are one of 4 built-in data types in Python used to store collections of data, the other 3 are Tuple, Set, and Dictionary, all with different qualities and usage. Lists are created using square brackets:

my_list = ["red", "white", "blue"]
print(my_list)

Output: ['red', 'white', 'blue']

List items are ordered, changeable, and allow duplicate values. List items are indexed, the first item has index [0], the second item has index [1] etc.
When we say that lists are ordered, it means that the items have a defined order, and that order will not change. If you add new items to a list, the new items will be placed at the end of the list.
The list is changeable, meaning that we can change, add, and remove items in a list after it has been created.
To determine how many items a list has, use the len() function:

my_list = ["red", "white", "blue", "green", "yellow"
print(len(my_list))

Output: 5

A list can contain strings, integers and boolean values. From Python's perspective, no matter the data types within the list, lists are defined as objects with the data type 'list':

my_list = ["red", 25, true, "ladder", 17.6]
print(type(my_list))

Output: <class 'list'>


Dictionaries

Dictionaries are used to store data values in key:value pairs. A dictionary is a collection which is unordered, changeable and does not allow duplicates. Dictionaries are written with curly brackets, and have keys and values:

my_dict = {
"make": "Chevy",
"model": "Nova",
"year": 1966
}
print(my_dict)

Output: {'make': 'Chevy', 'model': 'Nova', 'year': 1966}

Dictionary items are unordered, changeable, and does not allow duplicates. Dictionary items are presented in key:value pairs, and can be referred to by using the key name.

my_dict = {
"make": "Chevy",
"model": "Nova",
"year": 1966
}
print(my_dict["make"])

Output: Chevy


Conditional Statements

Below is an example of a Python if statement. If the statement after the "if" keyword is true, then the code below the if statement will be run. Python relies on indentation (whitespace at the beginning of a line) to define scope in the code. Other programming languages often use curly-brackets for this purpose. An if statement, without indentation will raise an error.


if statement:

n = 7
if n > 5:
    print(f'{n} is greater than 5')

Output: 7 is greater than 5

The else keyword catches anything which isn't caught by the preceding conditions.

if/else statement:

n = 2
if n > 5:
    print(f'{n} is greater than 5')
else:
    print(f'{n} is not greater than 5')

Output: 2 is not greater than 5

The elif keyword is pythons way of saying "if the previous conditions were not true, then try this condition".

elif statement:

n = 5
if n > 5:
    print(f'{n} is greater than 5')
elif n == 5:
    print(f'{n} is equal to 5')
elif n < 5:
    print(f'{n} is less than 5')
else:
    print(f'{n} might not be a number')

Output: 5 is equal to 5


Loops

Python has two primitive loop commands:

  • while loops
  • for loops

While loop - with the while loop we can execute a set of statements as long as a condition is true, or while it's true.

a = 1
while a < 5:
    print(a)
    a =+ 1

Output:
1
2
3
4

The code above stops after the fourth loop because the statement after the "while" keywork is no longer true. As you can see the variable a gets incremented by 1 every time the loop runs. If there is no code that is excecuted in the while loop that makes it false, the while loop will run forever and make your program crash. So there has to always be some was to exit the while loop.



For loop - A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc.

my_list = ['red', 'orange', 'yellow', 'green', 'blue', 'indego', 'violet']
for c in my_list:
    print(c)

Output
red
orange
yellow
green
blue
indego
violet

The for loop does not require an indexing variable to set beforehand.


Even strings are iterable objects, they contain a sequence of characters:

c = 'orange'
for l in c:
    print(l)

Output:
o
r
a
n
g
e

To loop through a set of code a specified number of times, we can use the range() function. The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number.

for x in range(5):
    print(x)

Output:
0
1
2
3
4


Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result.

#function definition with no arguments:
def greeting():
    print('Hello friend')

#calling the function from the program
greeting()

Output: Hello friend

Here is an example of a function that takes 1 parameter. Functions can take more than one parameter.

#function definition with 1 argument:
def greeting(name):
    print('Hello ' + name)

#getting user input and calling the function from the program
name = input("Enter your name: ")
greeting(name)

Output: Hello Kevin

Here is an exapmle of a function using return

#function definition with 1 argument and return stsement:
def greeting(name):
    result = 'Hello ' + name
return result

#getting user input and calling the function from the program
name = input("Enter your name: ")
print(greeting(name))

Output: Hello Kevin

You can just print the returned result, as shown above, or do other things with it.
