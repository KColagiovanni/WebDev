Palindrome Checker

A palindrome is a word or sentence that's spelled the same way both forward and backward, ignoring punctuation, case, and spacing.
Strings with varying formats, such as racecar, RaceCar, and race CAR among others will be passed through the function as well as special symbols, such as 2A3*3a2, 2A3 3a2, and 2_A3*3#A2.

Note: All non-alphanumeric characters (punctuation, spaces and symbols) were removed and everything is changed into the same case (lower or upper case) in order to check for palindromes.

Here are some exapmles of palindromes that will return true:

  • eye
  • _eye
  • race car
  • A man, a plan, a canal. Panama
  • never odd or even
  • My age is 0, 0 si ega ym.
  • 0_0 (: /-\ :) 0-0

Here are some exaples that will not return true:

  • not a palindrome
  • nope
  • almostomla
  • 1 eye for of 1 eye.
  • five|\_/|four

Roman Numeral Converter

Rules:

  • -In the Roman numeral system, the basic "digits" are the letters I, V, X, L, C, D, and M which represent the same numbers regardless of their position.

  • -Symbols are placed in order of value, starting with the largest values. When the higher numeral is placed before a lower numeral, the values of each Roman numeral are added.

  • -When smaller values precede larger values, the smaller values are subtracted from the larger values, and the result is added to the total.

  • -Do not repeat I, X, and C more than three times in a row. (Number 4 on a Roman numeral clock is usually written as IIII.)

    • -Symbols V, L, and D cannot appear more than once consecutively.

  • -Do not subtract a number from one that is more than 10 times greater: I may only precede V and X, X may only precede L and C, and C may only precede D and M.

Note: The largest number that can be represented in this notation is 3,999 (MMMCMXCIX), but since the largest Roman numeral likely to be required today is MMXXI (the current year) there is no practical need for larger Roman numerals. Prior to the introduction of Arabic numerals in the West, ancient and medieval users of the system used various means to write larger numbers. Read more about Roman Numerals on Wikipedia

Key:

  • I = 1
  • V = 5
  • X = 10
  • L = 50
  • C = 100
  • D = 500
  • M = 1000

Caesars Cipher

One of the simplest and most widely known ciphers is a Caesar cipher, also known as a shift cipher. In a shift cipher the meanings of the letters are shifted by some set amount. A common modern use is the ROT13 cipher, where the values of the letters are shifted by 13 places. Thus A ↔ N, B ↔ O and so on. This function has the added functionality of allowing the user to adjust the shift amount from 0 (no shift) to 25 (-1 shift). In addition to using the "Decode" button to get the result, the function will also update the value when then shift number value changes to make it easier to decode a phrase or word.

This program is showing a function which takes an encoded or decoded string as input and returns a decoded string that is shifted by a user defined amount. All letters will be converted to uppercase if they are not already.

Note: This function will decode a cipher or encode a regular phrase.

Here are some examples to try that are shifted by 13:

  • SERR CVMMN!
  • GUR DHVPX OEBJA SBK WHZCF BIRE GUR YNML QBT.
  • SERR YBIR?



Telephone Number Validator

You may fill out the form field any way they choose as long as it has the format of a valid US number. This function will validate or reject the US phone number based on any combination of the formats. The area code is required. If the country code is provided, you must confirm that the country code is 1.

Here are some examples of valid US phone numbers:

  • 1 555-555-5555
  • 1 (555) 555-5555
  • 5555555555
  • 555-555-5555
  • (555)555-5555
  • 1(555)555-5555
  • 1 555 555 5555
  • 1 456 789 4444

Here are some examples of invalid US phone numbers:

  • 555-5555
  • 5555555
  • 1 555)555-5555
  • 123**&!!asdf#
  • 55555555
